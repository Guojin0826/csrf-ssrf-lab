# CSRF & SSRF 演示平台 - 初始化功能说明

## 🔄 初始化按钮功能

### 功能位置
主页顶部，在警告提示框下方，有一个醒目的绿色按钮：
**"🔄 初始化演示环境"**

- 登录状态清除

#### 2. 论坛系统数据重置

- **xiaoming用户**：密码恢复到 123456
- **zhangsan用户**：密码恢复到 123456

- 登录状态清除

#### 3. CSRF基础演示数据重置
- **admin用户**：密码恢复到 admin123，邮箱恢复到 admin@example.com

### 使用场景

#### 攻击演示后重置
当您完成CSRF攻击演示后：
1. 银行账户余额可能已被修改
2. 论坛用户密码可能已被修改
3. CSRF演示用户信息可能已被修改

点击初始化按钮，可以一键恢复所有数据，方便进行下一轮演示。

#### 多次演示需求
在教学场景中，可能需要多次演示攻击过程：
- 第一次演示：展示攻击成功
- 点击初始化按钮重置环境
- 第二次演示：再次展示攻击过程
- 无需手动修改数据

### 操作流程

1. **点击按钮**：点击"🔄 初始化演示环境"按钮
2. **确认操作**：弹出确认对话框，点击"确定"
3. **等待执行**：系统执行重置操作（约1-2秒）
4. **查看结果**：显示成功消息，包含重置的用户数量
5. **自动消失**：成功消息3秒后自动消失

### 技术实现

#### 重置脚本（reset.php）
```php
// 清除所有session数据
session_destroy();

// 重置银行用户数据
$_SESSION['bank_users'] = [
    'zhangsan' => ['balance' => 50000.00, ...],
    'lisi' => ['balance' => 30000.00, ...],
    'wangwu' => ['balance' => 80000.00, ...]
];

// 重置论坛用户数据文件
file_put_contents('forum_users.json', json_encode($forumUsers));

// 重置CSRF用户数据文件
file_put_contents('users.json', json_encode($csrfUsers));
```

#### 前端交互（JavaScript）
```javascript
function resetEnvironment() {
    if (confirm('确定要重置演示环境吗？')) {
        fetch('reset.php')
            .then(response => response.json())
            .then(data => {
                // 显示成功消息
                // 3秒后自动消失
            });
    }
}
```

### 注意事项

1. **确认操作**：重置操作会清除所有数据，请确认后再执行
2. **登录状态**：重置后会清除所有登录状态，需要重新登录
3. **数据恢复**：所有数据恢复到初始状态，无法撤销
4. **多人使用**：如果是多人同时使用演示平台，重置会影响所有人
